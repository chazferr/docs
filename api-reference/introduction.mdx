## Overview

The Omnera API provides programmatic access to our trading platform, including Token Intelligence, trade execution, and portfolio management.

<Warning>
  The Omnera API is currently in private beta. Contact us to request access.
</Warning>

## Base URL

```
https://api.omnera.trade/v1
```

## Authentication

All API requests require authentication via Bearer token. Include your API key in the `Authorization` header:

```bash
curl https://api.omnera.trade/v1/tokens/analyze \
  -H "Authorization: Bearer YOUR_API_KEY"
```

<Note>
  Keep your API key secure. Never expose it in client-side code or public repositories.
</Note>

## Response Format

All responses are returned in JSON format:

```json
{
  "success": true,
  "data": {
    // Response data
  },
  "meta": {
    "requestId": "req_123abc",
    "timestamp": 1704067200
  }
}
```

## Error Handling

Errors return appropriate HTTP status codes with details:

```json
{
  "success": false,
  "error": {
    "code": "INVALID_TOKEN_ADDRESS",
    "message": "The provided token address is not valid"
  }
}
```

| Status Code | Description |
|-------------|-------------|
| 400 | Bad Request - Invalid parameters |
| 401 | Unauthorized - Invalid or missing API key |
| 429 | Rate Limited - Too many requests |
| 500 | Server Error - Something went wrong |
