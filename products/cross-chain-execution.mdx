Cross-chain trading has historically required users to manage sequential operations across multiple interfaces. Bridging introduces custody risk, wrapping introduces basis risk, and each additional transaction adds failure surface. The operational overhead eliminates most cross-chain opportunities before they can be acted upon.

Omnera abstracts this entirely. The user expresses intent. Omnera resolves the execution path. The user receives native assets on the destination chain without managing any intermediate steps.

## Execution Model

Omnera uses intent-based routing. When a user initiates a trade, the system determines the optimal path across supported chains. Execution is atomic: the trade either completes in full or does not execute. There is no partial state and no stuck funds.

Omnera does not act as a bridge. Assets are not held in custody during transit. The atomicity guarantee eliminates the class of failure modes associated with traditional cross-chain bridging.

## Supported Chains

Omnera currently supports Solana and Base. Additional chains are in active development. As each chain is integrated, opportunities on that chain surface in the feed automatically based on user relevance.
