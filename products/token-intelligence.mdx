## Using Token Intelligence

### Quick Search

1. Navigate to the Token Intelligence page or use the search bar
2. Paste a Solana token address or search by name
3. Receive your AI brief in approximately 3 seconds

### Understanding the Brief

Each Token Intelligence brief includes:

| Section | Description |
|---------|-------------|
| **Risk Score** | Overall risk assessment from 0-100 |
| **Contract Analysis** | Smart contract security findings |
| **Holder Metrics** | Distribution and concentration data |
| **Liquidity Status** | Pool health and lock status |
| **Social Pulse** | Community activity and sentiment |
| **AI Summary** | Plain-English summary of key findings |

## Best Practices

<Tip>
  Always use Token Intelligence before entering any trade, especially on newly launched tokens.
</Tip>

- **Verify the address**: Make sure you're analyzing the correct token contract
- **Check liquidity locks**: Unlocked liquidity can be pulled at any time
- **Review holder distribution**: High concentration in few wallets is a red flag
- **Monitor over time**: Run multiple checks as the token matures
