title: Get Started
description: How to access Omnera.
---

Omnera is in early access. Participation begins with a waitlist request at omnera.io.

## Waitlist

Early access invitations are distributed on a rolling basis as system capacity increases. There is no public launch date. Waitlist position and community participation both factor into invitation timing.

## Wallet Connection

On receiving an invitation, users connect a non-custodial wallet supporting Solana or Base. Omnera does not hold funds at any point. The wallet connection establishes identity for the behavioral model without granting Omnera access to assets.

## First Session

The behavioral model begins capturing data from the first session. It records what the user views, how long attention is held on each opportunity, and which positions are considered but not taken. There is no configuration step. The model initializes automatically and refines with each subsequent session.

## Early Access Scope

The feed, pre-trade intelligence, and cross-chain execution are available during early access. Feature availability may differ from the full product release.
