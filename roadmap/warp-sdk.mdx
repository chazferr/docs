# Developer SDK

The Omnera developer SDK will allow third-party applications to integrate cross-chain execution capabilities directly. Partners can offer multi-chain trading to their users without building or maintaining bridge infrastructure.

The SDK is planned for release following expanded chain support. A specific timeline has not been announced.

## Capabilities

The SDK exposes intent submission, quote aggregation across available routes, real-time execution status tracking, and webhook callbacks on trade completion. Partners submit cross-chain intents on behalf of their users. The same atomic execution guarantees that apply to direct Omnera trades apply to SDK-routed trades.

## Code Examples

### Get a Quote

```typescript
import { WarpSDK } from '@omnera/warp-sdk';

const warp = new WarpSDK({ apiKey: 'your-key' });

const quote = await warp.getQuote({
  sourceChain: 'solana',
  sourceToken: 'SOL',
  destChain: 'base',
  destToken: 'USDC',
  amount: '1000000000' // lamports
});
```

### Execute Intent

```typescript
const intent = await warp.submitIntent({
  quote: quote.id,
  userAddress: 'wallet-address',
  slippageTolerance: 0.5
});

// Track status
const status = await warp.getIntentStatus(intent.id);
```

## Use Cases

The SDK is designed for DEX aggregators that want to add cross-chain swap capability, wallet applications that want to enable cross-chain asset management, and DeFi protocols that want to support cross-chain deposits and withdrawals.
