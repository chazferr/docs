## SDK Features

### Core Capabilities

| Feature | Description |
|---------|-------------|
| Intent Submission | Submit cross-chain intents on behalf of users |
| Quote Aggregation | Get real-time quotes across all available routes |
| Status Tracking | Monitor intent execution status in real-time |
| Webhook Support | Receive callbacks on intent completion |

### Developer Experience

- **TypeScript/JavaScript SDK** with full type definitions
- **REST API** for server-side integrations
- **React Hooks** for frontend applications
- **Comprehensive documentation** and examples

## Integration Examples

### Basic Quote Request

```typescript
import { WarpSDK } from '@omnera/warp-sdk';

const warp = new WarpSDK({ apiKey: 'your-api-key' });

const quote = await warp.getQuote({
  sourceChain: 'solana',
  sourceToken: 'SOL',
  destChain: 'ethereum',
  destToken: 'ETH',
  amount: '1000000000' // 1 SOL in lamports
});

console.log(quote.expectedOutput); // ETH amount
console.log(quote.routes); // Available execution paths
```

### Submit Intent

```typescript
const intent = await warp.submitIntent({
  quote: quote.id,
  userAddress: 'user-wallet-address',
  slippageTolerance: 0.5 // 0.5%
});

// Track execution
const status = await warp.getIntentStatus(intent.id);
```

### React Hook

```typescript
import { useWarpQuote } from '@omnera/warp-sdk/react';

function SwapWidget() {
  const { quote, loading, error } = useWarpQuote({
    sourceChain: 'solana',
    sourceToken: 'SOL',
    destChain: 'base',
    destToken: 'USDC',
    amount: inputAmount
  });

  return (
    <div>
      {loading && <Spinner />}
      {quote && <QuoteDisplay quote={quote} />}
    </div>
  );
}
```
