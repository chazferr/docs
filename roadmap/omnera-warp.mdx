# Expanded Chain Support

Omnera currently supports cross-chain execution across Solana and Base. Additional chains are in active development.

## Rationale

Each chain added to the platform increases the total opportunity set available to the feed. More importantly, cross-chain behavioral data compounds the feed model. A user who trades spot on Solana and perpetual futures on Base generates a richer behavioral profile than a user confined to a single chain. Additional chains increase model resolution.

## Planned Integrations

Ethereum and Arbitrum are planned for integration. Additional L2 chains will follow. Specific timelines have not been announced. As each chain is integrated, opportunities on that chain will surface in the feed automatically based on user relevance.

## Intent-Based Routing

Cross-chain execution uses intent-based routing. The user specifies the desired outcome. The system determines the optimal execution path across available chains. A solver network competes to fill each intent, producing price competition that benefits the user.

Execution remains atomic regardless of chain count. The trade either completes in full or does not execute. There is no partial state.
