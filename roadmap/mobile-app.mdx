Omnera is building a native application for iOS and Android. The feed, pre-trade intelligence, and cross-chain execution will all be available on mobile.

## Design Principles

The mobile experience is built around the same behavioral feed model as the desktop interface. The pattern is scroll, evaluate, execute. All three actions occur within a single screen with no navigation between separate research and execution views.

The interface is optimized for single-hand use. Trade execution requires a single tap. Pre-trade intelligence surfaces inline without requiring the user to leave the feed context.

## Notifications

The mobile application extends the feed beyond active sessions. When the behavioral model identifies a high-relevance opportunity, a notification is delivered. Notifications are generated by the same model that drives the feed. They are not generic alerts.

Notification categories include opportunities matched to the user's profile, price movements on previously engaged assets, and risk signal changes on open or watched positions.

## Platform Status

Both iOS and Android builds are in active development. A release date has not been announced. Waitlist members will receive notification when the mobile application becomes available.
