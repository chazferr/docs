The Omnera mobile application is in development for iOS and Android. The feed, pre-trade intelligence, and cross-chain execution will be available in a native mobile interface.

## Design Principles

The mobile experience is built around the same behavioral feed model as the desktop interface. The interaction pattern is scroll, evaluate, execute. All three actions occur within a single screen. There is no navigation between separate research and execution views.

The interface is optimized for single-hand use. Trade execution requires a single tap. Pre-trade intelligence surfaces inline without requiring the user to leave the feed context.

## Notifications

The mobile application extends the feed beyond active sessions. When the behavioral model identifies an opportunity with high relevance to a user's profile, a notification is delivered. Notifications are generated based on the same behavioral model that drives the feed. They are not generic alerts.

Notification categories include feed-relevant opportunities, price movements on assets the user has previously engaged with, and risk signal changes on positions or watched assets.

## Platform Status

Both iOS and Android builds are in active development. Release timing has not been announced.
