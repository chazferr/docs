This roadmap reflects current development priorities. Features and timelines are subject to change without notice.

## Personalized Feed

The personalized feed is live on desktop. Behavioral data capture is active from session one. A mobile-native version of the feed is in development.

## Pre-Trade Intelligence

Pre-trade intelligence is live. Signal coverage is expanding as additional on-chain data sources are integrated. Current coverage includes wallet activity patterns, historical price behavior, and behavioral context from the user's feed profile.

## Cross-Chain Execution

Cross-chain execution is live on Solana and Base. Additional chain integrations are in progress. Each new chain added to the platform expands the opportunity set the feed can surface and compounds the behavioral model with cross-chain signal.

## Prediction Markets

Prediction market integration is in development. These features will surface information markets within the same feed as spot and perpetual futures positions. Availability will be limited to jurisdictions where participation is legally permitted.

Prediction market features are available where legally permitted. Users are responsible for ensuring compliance with applicable local laws and regulations before participating. This is not financial advice.

## Mobile Application

A native mobile application for iOS and Android is in development. The mobile experience is built around the same behavioral feed model used on desktop, optimized for single-hand interaction and one-tap execution.
